! = Criar forma basica do HTML


<h1> = Criar um Titulo dentro do "site"


<p> = Criar um paragrafo


<hr> = Criar uma linha horizontal


<br> = Faz a quebra de texto, é usado no lugar exato que você quer quebrar a linha


&-lt; e &-gt;(Sem o "-", é junto) = Serve para criar as <> das tags, porque caso você use normalmente, a tag funcionara, exemplo do <p>, se fazer sem o < e >, iria criar um parágrafo "lt" significa Less Than, menor que e "gt" Greater than, maior que, como os nomes dos sinais


Adicionando símbolos especiais basta colocat "&" e o nome do simbolo na frente, exemplos: ©(copyright), €(Euro), ↑(Seta para cima) e ↑(Seta para cima abreviada)
